# Rock Paper Scissors

A simple interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

This version runs directly in the browser with a clean user interface and score tracking. The first player to reach 5 points wins the game.

## ğŸ® How to Play

1. Open `index.html` in your browser.
2. Click Rock, Paper, or Scissors.
3. The computer makes a random choice.
4. Scores update automatically.
5. First to 5 points wins.
6. Click "Restart Game" to play again.
